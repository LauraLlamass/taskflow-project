# TaskFlow – Wireframe

## Diseño general de la aplicación

La aplicación TaskFlow está diseñada para ayudar a organizar tareas relacionadas con la escritura de un libro.

La interfaz se divide en tres partes principales:

### 1. Header
Contiene el título de la aplicación y el botón para cambiar entre modo claro y modo oscuro.

### 2. Área principal
Incluye:

- Buscador de tareas
- Formulario para añadir nuevas tareas
- Lista de tareas

Las tareas permiten:
- marcar como completadas
- eliminarse
- buscarse por texto

### 3. Panel lateral de estadísticas
Muestra información sobre las tareas:

- total de tareas
- tareas completadas
- tareas pendientes

También incluye ejemplos de tareas para ayudar al usuario a planificar su libro.
